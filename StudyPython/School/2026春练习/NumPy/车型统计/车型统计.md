# 本关任务
某汽车品牌4S店销售了 5 种车型（轿车、SUV、MPV、跑车、皮卡），记录了 2025 年四个季度每种车型的销量数据（单位：辆）。销售总监希望快速分析各车型的年度表现、各季度的总销量趋势、以及找出异常偏低的季度-车型组合，以便调整库存和营销策略。

从键盘录入一个 5×4 的嵌套列表构成numpy二维数组 sales，其中：
- 行索引 0,1,2,3,4 分别代表轿车、SUV、MPV、跑车、皮卡 五种车型
- 列索引 0,1,2,3 分别代表 第1季度、第2季度、第3季度、第4季度

例如：
```
[[152 142  64 156]
 [121  70 152 171]
 [124 137 166 149]
 [153 180 199 102]
 [ 51 137  87 179]]
```
请编写代码完成以下任务：
1. 车型年度统计：计算每个车型的全年总销量和平均季度销量，输出两个形状为 (5,) 的数组 model_total 和 model_avg。
2. 季度销售趋势：计算每个季度的所有车型总销量，输出形状为 (4,) 的数组 quarter_total。
3. 明星车型：找出每个季度的明星车型，即每个季度销量最高的车型。
4. 低销量预警：找出每个季度的最低销量车型，即每个季度销量最低的车型。

# 测试输入：
`[[152, 142, 64, 156], [121, 70, 152, 171], [124, 137, 166, 149], [153, 180, 199, 102], [51, 137, 87, 179]]`

```
车型总销量: [514 514 576 634 454]

各季度明星车型: ['跑车' '跑车' '跑车' '皮卡'] 
各季度最低销量车型: ['皮卡' 'SUV' '轿车' '跑车']
```
