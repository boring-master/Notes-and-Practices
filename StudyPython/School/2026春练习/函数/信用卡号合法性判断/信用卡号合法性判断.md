# 本关任务：
# 编写一个能判断信用卡号是否合法性的小程序。

# 信用卡号是否合法的判断规则为：
对给定的8位信用卡号码，如43589795，从最右边数字开始，隔一位取一个数相加，如5+7+8+3=23。  
将卡号中未出现在第一步中的每个数字乘2，然后将相乘的结果的每位数字相加。例如，对上述例子，未出现在第一步中的数字乘2后分别为（从右至左）18、18、10、8，则将所有数字相加为1+8+1+8+1+0+8=27。  
将上述两步得到的数字相加，如果得数个位为0，则输入的信用卡号是有效的。  

# 测试输入： 
```

1234567
43589795
q
```
