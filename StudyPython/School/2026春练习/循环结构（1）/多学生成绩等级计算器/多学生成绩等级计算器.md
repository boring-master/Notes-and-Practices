# 本关任务：
# 编写一个能输入多个学生成绩并计算等级的计算器。
# 成绩计算如下：
总成绩按平时1，平时2，期中，期末，各成绩百分比占20%，30%，10%，40%计算
# 成绩等级判断如下：
总评成绩90到100分，成绩等级“优秀”，80-90分的成绩等级“良好”，70-80的“中等”，60-70分的“及格”，60以下的“不及格”，若学生期末成绩低于30分，不计算平时成绩，则总评成绩为期末成绩，成绩等级“不及格”。
# 测试输入
```
hzh001  
90,100  
89,99  
hzh002  
90,120  
90,87  
hzh002  
90,100  
90,87  
exit
```
# 预期输出
```
输入错误: 成绩必须在0-100分之间,请重新输入  
最终成绩汇总：  
hzh001: 96.5分 (优秀)  
hzh002: 91.8分 (优秀)  
```
