# 本关任务：
给定一个图片文件test.png，读取指定字节的内容并保存到新的图片文件中。
# 二进制文件读写
二进制文件一般是可执行程序、图形、图像、声音文件等，一般不能直接读懂二进制文件，只有通过相应的软件才能将其显示出来。

现有一个图片文件example.png
1. 读文件
    ```Python
    with open("example.png","rb") as f:
        print(f.read())   #读取全部字节
    with open("example.png","rb") as f:
        print(f.read(10))  #只读取10个字节 
    ```
    > 注意：输出结果因不同图片文件而异
2. 写文件
    ```Python
    with open("example.png","rb") as f,open("new.png","wb") as fb:
        data = f.read(20) #从example图片读20字节
        fb.write(data) #写入new图片
    ```
    > 注意：写入二进制文件时，只能写入 bytes 类型的数据
# 测试输入：`10`
# 预期输出：`b'\x89PNG\r\n\x1a\n\x00\x00'`
