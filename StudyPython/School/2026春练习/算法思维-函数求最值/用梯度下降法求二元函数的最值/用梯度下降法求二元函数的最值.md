# 本关任务：
用梯度下降法求二元函数$f(x,y)=x^3-y^3+3x^2+3y^2−9x$的最小值。当最小值处的函数的两个偏导数值的绝对值都小于1e-6时结束迭代,设置最大迭代次数10000，如果迭代结束都没有达到精度要求，则输出False,否则以保留8位小数的方式输出最小值以及此时的x值和y值，以及迭代次数。  
> 注意：请按顺序更新x和y值，且使用更新后的x值来求函数对y的偏导。

# 二元函数的梯度下降法公式：
$(x_{k+1},y_{k+1})=(x_k,y_k)-\eta *(\frac{\partial f}{\partial x},\frac{\partial f}{\partial y})$

# 测试输入1：
```
3, 4
0.1
```
# 预期输出1：`迭代次数14,x=1.00000000,y=0.00000014,f(x,y)=-5.00000000`
# 测试输入2：
```
3, 4
0.2
```
# 预期输出2：`False`
