# 本关任务：
# 编写一个能将3位数以内的整数翻译成英文的程序。

# 如何实现整数的翻译
# 用函数封装方式来完成，试题中已定义了三个函数，请完成这三个函数：
1. unit_to_word将0～9的数字转换成英文，并返回转换后的英文；
2. tens_to_word利用unit_to_word，将10～19、以及20～99的十位部分数字转换成英文，并返回转换后的英文；
3. hundreds_to_word利用unit_to_word、tens_to_word进行转换，并返回转换后结果。
> 请注意，得到的英文字符串前后没有空格。
# 测试输入1：`121`
# 预期输出1：`121 ==> one hundred and twenty one`

# 测试输入2：`0`
# 预期输出2：`zero`
