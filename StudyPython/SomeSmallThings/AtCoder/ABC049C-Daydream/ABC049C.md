# 问题陈述
你得到一个由小写英文字母组成的字符串$S$。另一个字符串$T$最初是空的。判断是否可以通过任意次数的以下操作来使得$T$变为$S=T$：
- 在$T$的末尾添加以下中的一个：`dream`、`dreamer`、`erase`和`eraser` 
# 限制条件
- $1 \leq |S| \leq 10^5$
- $S$由小写英文字母组成
# 输入
输入数据采用标准输入方式提供，格式如下：
```
S
```
# 输出结果
如果有可能得到$S=T$，则输出`YES`。否则，输出`NO`
# 示例输入1:
```
erasedream
```
# 示例输出 1：
```
YES
```
# 示例输入 2：
```
dreamerer
```
# 示例输出 2：
```
NO
```
