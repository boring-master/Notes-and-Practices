# 问题陈述
AtCoDeer这只鹿正在二维平面上进行旅行。在他的计划中，他将从点$(0,0)$在时间$0$出发，然后对于每个$i$在$1$和$N$之间（包括$1$和$N$），他将在时间$t_i$访问点$(x_i,y_i)$  

如果AtCoDeer在时间$t$位于点$(x,y)$，那么在时间$t+1$他可以位于以下点之一：$(x+1,y)$、$(x−1,y)$、$(x,y+1)$和$(x,y−1)$。请注意他不能停留在原地。确定他是否能够完成他的计划
# 限制条件
- $1 \leq N \leq 10^5$
- $0 \leq x_i \leq 10^5$
- $0 \leq y_i \leq 10^5$
- $1 \leq t_i \leq 10^5$
- $t_i \leq t_{i+1}(1\leq i\leq N-1)$
- 所有输入值都是整数
# 输入
输入数据采用标准输入方式提供，格式如下：
```
N
t1 x1 y1
t2 x2 y2
.
.
tN xN yN
```
# 输出结果
如果AtCoDeer能够完成他的计划，打印`Yes`；如果他不能，打印`No`
# 示例输入 1：
```
2
3 1 2
6 1 1
```
# 示例输出 1：
```
Yes
```
# 示例输入 2：
``` 
2
5 1 1
100 1 1
```
# 示例输出 2：
``` 
No
```
